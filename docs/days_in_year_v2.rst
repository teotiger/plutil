DAYS_IN_YEAR Function (2)
=========================

This function returns the number of days in the year.

Syntax
------

.. code-block:: SQL

  PLUTIL.DAYS_IN_YEAR (
    a_year  IN  NUMBER)
  RETURN NUMBER;

Parameter
---------

===================== =====================
Parameter             Description
===================== =====================
A_YEAR                The year to check.
===================== =====================
