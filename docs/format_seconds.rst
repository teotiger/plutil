FORMAT_SECONDS Function
=======================

This function formats the number of seconds.

Syntax
------

.. code-block:: SQL

  PLUTIL.FORMAT_SECONDS (
    p_seconds  IN  NUMBER)
  RETURN VARCHAR2;

Parameter
---------

===================== =====================
Parametername         Parameterbeschreibung
===================== =====================
P_SECONDS             The number of seconds.
===================== =====================
