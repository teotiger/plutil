IS_LEAP_YEAR Function (1)
=========================

This function checks if the date is a leap year or not.

Syntax
------

.. code-block:: SQL

  PLUTIL.IS_LEAP_YEAR (
    a_year  IN  DATE)
  RETURN BOOLEAN;

Parameter
---------

===================== =====================
Parameter             Description
===================== =====================
A_DATE                The date to check.
===================== =====================
