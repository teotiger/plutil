IS_LEAP_YEAR Function
=====================

This function checks if the date is a leap year or not.

Syntax
------

.. code-block:: SQL

  PLUTIL.IS_LEAP_YEAR (
    p_year  IN  DATE)
  RETURN PL/SQL BOOLEAN;

Parameter
---------

===================== =====================
Parameter             Description
===================== =====================
A_YEAR                The date to check.
===================== =====================
