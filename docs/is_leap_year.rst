IS_LEAP_YEAR Function
=====================

This function checks if the date is a leap year or not.

Syntax
------

.. code-block:: SQL

  PLUTIL.IS_LEAP_YEAR (
    a_date  IN  DATE)
  RETURN PL/SQL BOOLEAN;

Parameter
---------

===================== =====================
Parameter             Description
===================== =====================
A_DATE                The date to check.
===================== =====================
