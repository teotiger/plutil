HEX_TO_RGB Function
===================

This function converts a hexadecimal color value into a RGB color value.

Syntax
------

.. code-block:: SQL

  PLUTIL.HEX_TO_RGB (
    a_hex  IN  VARCHAR2)
  RETURN VARCHAR2;

Parameter
---------

===================== =====================
Parameter             Description
===================== =====================
A_HEX                 A valid hexadecimal color value.
===================== =====================
