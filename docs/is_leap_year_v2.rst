IS_LEAP_YEAR Function
=====================

This function checks if the year is a leap year or not.

Syntax
------

.. code-block:: SQL

  PLUTIL.IS_LEAP_YEAR (
    p_year  IN  BINARY_INTEGER)
  RETURN PL/SQL BOOLEAN;

Parameter
---------

===================== =====================
Parameter             Description
===================== =====================
A_YEAR                The year in the format YYYY.
===================== =====================
