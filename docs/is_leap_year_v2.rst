IS_LEAP_YEAR Function (2)
=========================

This function checks if the year is a leap year or not.

Syntax
------

.. code-block:: SQL

  PLUTIL.IS_LEAP_YEAR (
    a_year  IN  PLS_INTEGER)
  RETURN BOOLEAN;

Parameter
---------

===================== =====================
Parameter             Description
===================== =====================
A_YEAR                The year in the format YYYY.
===================== =====================
