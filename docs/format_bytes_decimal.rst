FORMAT_BYTES_DECIMAL Function
=============================

This function formats the number of bytes. Base is 10.

Syntax
------

.. code-block:: SQL

  PLUTIL.FORMAT_BYTES_DECIMAL (
    a_bytes   IN NUMBER)
  RETURN VARCHAR2;

Parameter
---------

===================== =====================
Parameter             Description
===================== =====================
A_BYTES               The number of bytes.
===================== =====================
