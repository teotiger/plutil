DAYS_IN_MONTH Function
======================

This function returns the number of days in the month.

Syntax
------

.. code-block:: SQL

  PLUTIL.DAYS_IN_MONTH (
    a_month  IN  DATE)
  RETURN NUMBER;

Parameter
---------

===================== =====================
Parameter             Description
===================== =====================
A_MONTH               The date to check.
===================== =====================
