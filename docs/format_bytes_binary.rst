FORMAT_BYTES_BINARY Function
============================

This function formats the number of bytes. Base is 2.

Syntax
------

.. code-block:: SQL

  PLUTIL.FORMAT_BYTES_BINARY (
    a_bytes   IN NUMBER)
  RETURN VARCHAR2;

Parameter
---------

===================== =====================
Parameter             Description
===================== =====================
A_BYTES               The number of bytes.
===================== =====================
