DAYS_IN_YEAR Function (1)
=========================

This function returns the number of days in the year.

Syntax
------

.. code-block:: SQL

  PLUTIL.DAYS_IN_YEAR (
    p_year  IN  DATE)
  RETURN NUMBER;

Parameter
---------

===================== =====================
Parameter             Description
===================== =====================
A_YEAR                The date to check.
===================== =====================
